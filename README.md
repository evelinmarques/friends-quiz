# 💻 Como rodar o projeto:
Após clonar o projeto, acesse ele via seu terminal e execute os comandos:
```
yarn
```

```
yarn dev
```
Em seguida acesse no seu navegador:

```
http://localhost:3000/
```
# 🚀 Tecnologias:
* React.js
* Next.js
* Styled Components
* Lottie
* Babel
